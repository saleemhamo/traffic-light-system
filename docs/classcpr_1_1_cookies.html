bool cpr::Cookies::encode {true}

Should we URL-encode cookies when making a request. Based on RFC6265, it is recommended but not mandatory to encode cookies.

To maximize compatibility with user agents, servers that wish to store arbitrary data in a cookie-value SHOULD encode that data, for

example, using Base64 [RFC4648].

Source: RFC6265 (https://www.ietf.org/rfc/rfc6265.txt)
