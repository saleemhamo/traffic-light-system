Detailed Description

Defines the Logger class which provides logging functionalities both locally and to Firestore.

This class encapsulates methods for initializing, starting, stopping, and logging messages to a local file and Google Firestore. It uses asynchronous logging with multiple worker threads to manage writing operations without blocking the main application flow.
