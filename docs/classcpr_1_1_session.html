◆ ResponseStringReserve() [1/2]

void cpr::Session::ResponseStringReserve ( size_t size)

Attempt to preallocate enough memory for specified number of characters in the response string. Pass 0 to disable this behavior and let the response string be allocated dynamically on demand.

Example: cpr::Session session; session.SetUrl(cpr::Url{"http://xxx/file"}); session.ResponseStringReserve(1024 * 512); // Reserve space for at least 1024 * 512 characters cpr::Response r = session.Get();
