Detailed Description

template<typename T, bool isCancellable = false>
class cpr::AsyncWrapper< T, isCancellable >

A class template intended to wrap results of async operations (instances of std::future<T>) and also provide extended capablilities relaed to these requests, for example cancellation.

The RAII semantics are the same as std::future<T> - moveable, not copyable.
