Detailed Description

Controls a buzzer connected to a GPIO pin on a Raspberry Pi.

This file contains the implementation of the Buzzer class, which provides methods to turn a buzzer on and off. The class abstracts the details of using the pigpio library (or its stub for non-Raspberry Pi or non-ARM Linux environments) to manage GPIO pins.

If running on a typical Raspberry Pi OS and ARM architecture, the pigpio library is used. Otherwise, a stub implementation is utilized.
